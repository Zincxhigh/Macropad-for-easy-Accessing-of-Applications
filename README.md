# Macropad-for-easy-Accessing-of-Applications
Just a simple macropad which can be used for proper swipping and navigation!


A macropad with 6 switches 2 for navigation through virtual desktops and 4 for 
daily use application opening with an OLED for Tracking Time to be punctual.


:: THINGS IN USE ::

(<1>) = A microcontroller (MOUDLE-SEEEDUINO-XIAO) a 14-pin microcontroller with regular switches In a matrix System


(<2>) = The Conn_01x04 is a powers source for an I2C OLED display (0.96)


(<3>) = Also using a rotary Encoder for volume increase and decrease with onside button when clicked makes the sound mute
